INAPP INVENTORY DASHBOARD

INSTALLATION :

1. Ouvrir le dossier dans VS Code

2. Ouvrir le terminal

3. Exécuter la commande :

   npx serve

4. Ouvrir dans le navigateur :
   http://localhost:3000

IMPORTANT :

* Ne pas ouvrir index.html directement
* Une connexion internet est requise (CDN Bootstrap et ApexCharts)

STRUCTURE :

* index.html → page principale
* assets/css → styles
* assets/js → scripts
* assets/images → images

PERSONNALISATION :

* Modifier index.html pour changer l’interface
* Modifier style.css pour le design
* Modifier chart.js pour les graphiques
